# Duelist's Forge: The Card Crafter 3D / El Forjador de Cartas 3D

Welcome to **Duelist's Forge**! This is a bilingual, interactive web application designed to create, visualize, and download custom Yu-Gi-Oh! style cards with an immersive 3D flip effect.

¡Bienvenido a **Duelist's Forge**! Esta es una aplicación web interactiva y bilingüe diseñada para crear, visualizar y descargar cartas personalizadas al estilo Yu-Gi-Oh! con un efecto de giro 3D inmersivo.

## New Features / Nuevas Características

- **Improved Language Selector / Selector de Idiomas Mejorado**: 
  - Now shows the current language with flag emojis (🇺🇸 English / 🇲🇽 Español).
  - Ahora muestra el idioma actual con emojis de banderas (🇺🇸 English / 🇲🇽 Español).
- **Download Card / Descargar Carta**:
  - Export your custom card as a high-quality PNG image directly from the sidebar.
  - Exporta tu carta personalizada como una imagen PNG de alta calidad directamente desde el sidebar.
- **Dynamic Star Generation / Generación Dinámica de Estrellas**:
  - Stars are generated via JavaScript within a single optimized `<span>` with a dedicated CSS class.
  - Las estrellas se generan vía JavaScript dentro de un único `<span>` optimizado con una clase CSS dedicada.

## Features / Características

- **Dynamic Sidebar / Sidebar Dinámico**: 
  - Fields automatically hide or show based on the selected card type (e.g., hiding levels for Spells).
  - Los campos se ocultan o muestran automáticamente según el tipo de carta (ej: ocultar niveles para Magias).
- **Responsive & Fixed Layout / Diseño Responsivo y Corregido**:
  - Optimized grid to prevent page overflow and ensure smooth navigation on all devices.
  - Grid optimizado para evitar desbordamientos y asegurar una navegación fluida en todos los dispositivos.
- **Bilingual Interface / Interfaz Bilingüe**:
  - Support for both English and Spanish in the UI and documentation.
  - Soporte para inglés y español en la interfaz y documentación.
- **Custom Image Upload / Subida de Imagen Personalizada**:
  - Styled button to upload your own card art.
  - Botón estilizado para subir tu propio arte de carta.
- **3D Flip Effect / Efecto de Giro 3D**:
  - Realistic card rotation on hover or keyboard focus.
  - Rotación realista de la carta al pasar el ratón o usar el foco del teclado.
- **Accessibility / Accesibilidad**:
  - Full keyboard support (use **Tab** to flip the card).
  - Soporte completo de teclado (usa **Tab** para girar la carta).
  - ARIA labels for screen readers.
  - Etiquetas ARIA para lectores de pantalla.

## Technologies Used / Tecnologías Utilizadas

- **HTML5**: Semantic structure / Estructura semántica.
- **CSS3**: Grid/Flexbox layout, 3D transformations / Layout Grid/Flexbox, transformaciones 3D.
- **JavaScript (Vanilla)**: DOM manipulation and dynamic logic / Manipulación del DOM y lógica dinámica.
- **html2canvas**: For card image exporting / Para la exportación de la imagen de la carta.

## How to Use / Cómo Usar

1. Open `index.html` in your browser / Abre `index.html` en tu navegador.
2. Use the sidebar to customize your card / Usa el sidebar para personalizar tu carta.
3. Use the **Tab** key to focus and flip the card / Usa la tecla **Tab** para enfocar y girar la carta.
4. Upload your own art and click **Download Card** to save your creation / Sube tu propio arte y haz clic en **Descargar Carta** para guardar tu creación.
