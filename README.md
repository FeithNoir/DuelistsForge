# Duelist's Forge: Card Crafter 3D

¡Bienvenido a **Duelist's Forge**! Esta es una aplicación web interactiva diseñada para crear y visualizar cartas personalizadas al estilo Yu-Gi-Oh! con un efecto de giro 3D inmersivo.

## Características

- **Diseño Responsivo**: Utiliza CSS Grid para un diseño fluido en diferentes tamaños de pantalla.
- **Personalización en Tiempo Real**: Cambia el nombre, atributo, tipo, nivel y descripción de la carta y observa los cambios instantáneamente.
- **Subida de Imágenes**: Carga tu propio arte para la carta desde tu dispositivo.
- **Efecto 3D**: La carta gira al pasar el ratón o al recibir el foco del teclado.
- **Accesibilidad**: 
  - Navegación completa por teclado.
  - Atributos `aria` para lectores de pantalla.
  - Contraste de colores optimizado.
  - Semántica HTML5 adecuada.

## Tecnologías Utilizadas

- **HTML5**: Estructura semántica.
- **CSS3**: Layout con Grid y Flexbox, transformaciones 3D.
- **JavaScript (Vanilla)**: Lógica de actualización del DOM y manejo de archivos.

## Cómo Usar

1. Abre `index.html` en tu navegador preferido.
2. Usa el panel lateral izquierdo para personalizar los detalles de tu carta.
3. Haz clic en la carta o usa la tecla `Tab` para enfocarla y ver el reverso.
4. Para añadir una imagen, usa el botón de "Imagen de Arte".
